# Wallid Pay By Bank for WooCommerce

Wallid Plugin enables merchants to accept account-to-account payments in WooCommerce using Open Banking. Customers pay directly from their bank account to the merchant’s bank account, without using debit or credit cards.

## Installation

1. Download the Wallid Pay by Bank plugin as a ZIP file.
2. In your WordPress admin dashboard, go to Plugins → Add New → Upload Plugin.
3. Upload the ZIP file, install the plugin, and activate it.

## Account Setup

4. Create a merchant account at https://www.getwallid.co.uk.
5. Once your profile is complete, contact Wallid via live chat in the Merchant Portal or by email at support@corp.wallid.co to request account activation.
6. After verification, Wallid will provide your Client ID and Secret Key. Account activation may take several hours.

## Plugin Configuration

7. In your WordPress admin dashboard, go to WooCommerce → Settings → Payments → Wallid Pay by Bank.
8. Enter your Client ID and Secret Key, save the settings, and enable the payment method.

## Testing

9. Place a test order with a small amount to confirm that payments are processed correctly.
10. (Optional) Test the instant refund flow for the test payment via the Merchant Portal at https://www.getwallid.co.uk
